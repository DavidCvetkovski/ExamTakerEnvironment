#Q Explain the difference between Type I and Type II errors in hypothesis testing. Provide one real-world example of each.
TYPE: ESSAY
LEVEL: Analyze
DIFFICULTY: Hard
POINTS: 5
SUBJECT: Hypothesis Testing

MODEL_ANSWER:
A Type I error (false positive) occurs when the null hypothesis is true but is incorrectly rejected. Example: concluding a drug is effective when it is not.

A Type II error (false negative) occurs when the null hypothesis is false but is not rejected. Example: concluding a drug has no effect when it actually does.

The probability of a Type I error is alpha (significance level); Type II error probability is beta.
END_MODEL_ANSWER
